# Лабиринт Сокровищ

Текстовая приключенческая игра в жанре "лабиринт", где игрок исследует подземелья, решает загадки и ищет сокровища.

## Особенности

-  Система комнат с разными описаниями и предметами
-  Интерактивные загадки с альтернативными ответами
-  Система инвентаря и использования предметов
-  Случайные события и ловушки
-  Логика доступа к комнатам с помощью ключей
-  Интуитивные команды на английском

## Установка

```bash
# Установка зависимостей
make install
# или
poetry install


# Запуск игры
make project
# или

# Управление
Основные команды:

look - осмотреть текущую комнату

go [direction] - перемещение

north/south/east/west - перемещение (сокращенная форма)

take [item] - подобрать предмет

inventory - просмотр инвентаря

solve - решить загадку/открыть сундук

help - показать все команды

quit - выйти из игры
